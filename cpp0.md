# C++简介

大家好，这里是 **C++** 语言专题。C++是一门相对古老的语言，又是一门非常经典的语言。第三次工业革命以降，C++作为一门基础编程语言，支撑着信息产业界的繁荣盛世。视线可及的未来，我们很难看到它过时而退出历史舞台的一天。关于C++语言的优秀教程实在不胜枚举，这里让我们从一个初学者的视角，对这门语言一探究竟。

## 1. C++ 的来历

如果说西方思想史是对柏拉图的一系列脚注[<font color=blue> 怀特海,A.N.,英国数学家、哲学家 </font>]，那么可以把现代主流编程语言理解为对C语言的一系列优化、改进和拓展。C语言伴随UNIX操作系统的诞生而出现，是信息技术发展史中的重要事件。C语言既高效又灵活，有清晰严格的标准，支持跨平台移植。C语言很适合面向过程的朴素编程思想，但是随着编程理念的发展，提出了面向过程的编程思想，为了适应新潮流，C++语言应运而生。将面向对象编程思想中稳定性、重用性和扩展性的优势，从理论层面落地，大大提升了行业的生产力水平。 C++是C语言的超集，任何合法的C代码都是合法的C++代码。

## 2. C++ 的组成
标准的C++由三部分组成。 
- **C++内核**  提供了基础的变量、数据类型和常量。
- **C++标准库**  提供了大量函数，包含科学计算、文件操作、字符串操作等功能。
- **标准模版库**  提供了大量的方法，用于数据结构的操作。

## 3. C++ 的用武之地

为什么腾讯服务器端用C++而阿里巴巴用JAVA呢？ 可以在企业初期聚焦的产品找到答案，腾讯以qq起家更偏向通信一些，而阿里巴巴“让天下没有难做的生意”的初心更偏向erp系统一些，在这两个领域里C++和JAVA各有所长。如今腾讯的主要盈利点是游戏业务，而电商业务是阿里巴巴的创收支柱，这也从侧面体现了各自不同技术栈的储备和积累。业务造就了队伍，队伍又决定了传统。下面来看看C++语言适用的场景吧。

### 3.1 操作系统底层开发
  无论你使用苹果电脑、兼容机、智能手机还是嵌入式设备，无论你使用Windows操作系统、Linux操作系统、Android操作系统，还是Mac OS，你都在间接地使用C++，因为这些系统的主要用户接口是使用 C++ 编写的。毫无疑问，进行操作系统底层开发的工作也离不开C++语言。

### 3.2 桌面应用
  当你准备在Windows或Linux的某个发行版开发一款桌面应用，你会面临技术选型的问题。使用WIN32 API还是MFC，采用GTK+还是QT框架。无论结论如何，都需要你首先掌握C++语言。

### 3.3 电脑游戏
  电脑游戏往往需要进行频繁高效的人机交互，才能保证玩家丝滑流畅的游戏体验，这个问题在网游中更加凸显。C++语言以其高效灵活的特性，满足了游戏玩家所提出的要求，因此被游戏开发者们所青睐。

### 3.4 通信行业
  通信行业对性能和时效性有很高的要求。同时涉及各种底层通信协议及数据帧的处理，这些工作同样需要C++语言的高效性作为支撑。

### 3.5 安全领域
  因为C++语言和操作系统间紧密的关系，信息安全和黑客攻防也是C++大显身手的舞台。卡巴斯基、360安全卫士、远程控制软件、木马、外挂等等95%以上都是用C/C++编写。

### 3.6 科研领域
  在自然科学学科的科研攻关中，计算机是必不可少的计算和推衍工具。大量计算密集型工作甚至需要大型计算机和计算集群的帮助。C++语言也以其高效性，受到了很多科研人员的青睐。
